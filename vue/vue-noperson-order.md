## vue 无人点餐

###  环境搭建
1. 安装node

2. 搭建脚手架
 
 npm install -g vue-cli

 3. 创建项目

 vue init webpack vue-noperson-order

 cd vue-noperson-order

 npm i

 npm run dev


4. 另一种创建项目的方式

 vue init webpack-simple vue-noperson-order2

 cd vue-noperson-order

 npm i

 npm run dev
